# @hectoday/http Examples

Example applications demonstrating @hectoday/http usage across different runtimes.

## Deno Example

Full-featured example with validation, guards, and middleware.

```bash
cd deno
deno task dev
```

Visit: http://localhost:8000

## Bun Example

Minimal example for Bun runtime.

```bash
cd bun
bun install
bun run dev
```

Visit: http://localhost:3000

## Quick Start

From the root directory:

```bash
# Run Deno example
deno task example

# Run Bun example
deno task example:bun
```
